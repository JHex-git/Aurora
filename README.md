# Aurora
A toy OpenGL Render Engine.

## Platform support
| Platform | CI |
|----------|:--:|
| Windows | <p>[![MSVC CI](https://github.com/JHex-git/Aurora/actions/workflows/MSVC.yml/badge.svg)](https://github.com/JHex-git/Aurora/actions/workflows/MSVC.yml)</p> |

## Feature
* Phong shading
* Skybox
* Outline
* Shadow mapping
* Bounding Volume Hierarchy

## How to build
Only MSVC is tested. Run `MSVC_build.bat` to build the project.
